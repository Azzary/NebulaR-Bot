# Installation

Vous pouvez installer NebularBot en suivant ces étapes simples :

1. Rendez-vous sur le site [NebularBot.com/install](https://NebularBot.com/install).
2. Connectez-vous à votre compte ou inscrivez-vous si vous n'en avez pas déjà un.
3. Téléchargez le bot à partir du lien de téléchargement fourni.
4. Une fois le téléchargement terminé, exécutez le fichier d'installation et suivez les instructions à l'écran.
5. Après l'installation, lancez NebularBot et connectez-vous avec vos identifiants.
6. Vous êtes prêt à utiliser NebularBot !

N'oubliez pas que vous devez avoir un compte NebularBot pour pouvoir utiliser le bot. Si vous n'en avez pas encore, rendez-vous sur [NebularBot.com](https://NebularBot.com) et inscrivez-vous gratuitement.
